## Stepchart Analyzer

(WIP) A tool to analyze and visualize 4 panel rhythm game charts and give users insight into the patterns in a chart, the distribution throughout the chart, etc.

## Usage

1. install node.js
2. install yarn
3. yarn install
4. yarn dev
5. http://localhost:3000
